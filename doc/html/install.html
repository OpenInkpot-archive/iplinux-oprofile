Installation

First you need to build oprofile and install it. The holy trinity of ./configure, make, make install is all you need, but note these arguments to ./configure :

--with-cc

Use this to specify a C compiler that you used to compile the kernel if necessary (e.g. kgcc).

--enable-filter

Enable process id and process group filtering, as discussed in FIXME.

You'll need to have a configured kernel source for the current kernel to build the module. Also note you need to enable the CONFIG_X86_UP_IOAPIC or CONFIG_X86_UP_APIC options in your kernel configuration. Which one is available depends on kernel version (as of the time of writing, the second is only available in the ac series, but is to be preferred).
